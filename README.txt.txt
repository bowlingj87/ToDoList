Description:
This project is a functional To-Do List Application using Python from scratch. This application contains Python concepts such as syntax, data types, control structures, functions, and error handling, all while creating a practical, interactive command-line application.

Usage:
Run the main Python script 

Features:

- View existing tasks
- Delete tasks
